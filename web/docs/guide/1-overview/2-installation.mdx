:::info[整包 vs 单包]
`kcuf-ui` 仅仅是一个集合，所有的包实际上都可以独立安装，使用了 [Tree-shaking](https://developer.mozilla.org/en-US/docs/Glossary/Tree_shaking) 的情况，两者在最后打包的时候，没太大区别。
:::

## 使用要求

### Peer dependencies

以下为 [Peer dependencies](https://docs.npmjs.com/cli/v11/configuring-npm/package-json#peerdependencies)，需要自行安装：

* [react ≥ 18.x](https://www.npmjs.com/package/react)
* [react-dom ≥ 18.x](https://www.npmjs.com/package/react-dom)
* [styled-components ≥ 6.x](https://www.npmjs.com/package/styled-components)

### 对应用的要求

> 没有任何要求。

不少组件库，会要求有个设置环节，比如在应用外套一层它的 `Provider`，`kcuf-ui` 不需要，开箱即用。

### 对构建的要求

> 没有任何要求。

不少组件库，可能会要求使用方在其构建的时候加这加那，否则就可能跑起来有问题，甚至跑起来直接抛错。
