<ComponentBrief name="InputSwitch">
  切换 `true/false` 状态，可以认为是 `<input type="checkbox" />` 的变体。
</ComponentBrief>

## Live Demo

```tsx live
<InputSwitch defaultValue={true} />
```

## 何时使用

* 在两种状态之间切换时

## 组件构成

```text
<InputSwitch> --- +
| [knob] [label?] |
+ -- </InputSwitch>
```

## 代码演示

### 最简使用

```tsx live
<InputSwitch />
```

### 禁用

```tsx live
<InputSwitch label="disabled" disabled />
```

### 受控

```tsx live
function DemoControllable(): ReactElement {
  const [stateValue, setStateValue] = useState(true);
  const handleNativeInputChange = useCallback((e: ChangeEvent<HTMLInputElement>) => {
    setStateValue(e.target.checked);
  }, [setStateValue]);
  
  return <>
    <InputSwitch {...{
      label: '非受控',
      onChange: setStateValue
    }} />
    <InputSwitch {...{
      label: '受控 1',
      value: stateValue,
      onChange: setStateValue
    }} />
    <InputSwitch {...{
      label: '受控 2',
      value: stateValue,
      onChange: setStateValue
    }} />
    <input {...{
      type: 'checkbox',
      checked: stateValue,
      onChange: handleNativeInputChange
    }} />
    <input {...{
      type: 'radio',
      checked: stateValue,
      onChange: handleNativeInputChange
    }} />
  </>;
}
```

## API Reference

### import

```tsx
import {
  InputSwitch,
  InputSwitchRef,
  InputSwitchProps,
  InputSwitchType
} from 'kcuf-ui';

// 也可单包引入
import InputSwitch, {
  InputSwitchRef,
  InputSwitchProps,
  InputSwitchType
} from '@kcuf-ui/rc-input-text';
```

### Props

支持其所有 [input props](https://react.dev/reference/react-dom/components/input#props)，除了：

* `type`，固定为 `'text'`

| 属性 | 类型 | 说明 |
| --- | --- | --- |
| `value` | `boolean` | 受控值 |
| `defaultValue` <TagDefault>false</TagDefault> | `boolean` | 默认值 |
| `disabled` <TagDefault>false</TagDefault> | `boolean` | 是否禁用 |
| `label` | `string \| ReactElement` | 显示 Label |
| `onChange` | `(value: boolean) => void` | 用户操作引起的值变化时的回调 |

### Events

| 事件 | 类型 | 说明 |
| --- | --- | --- |
| `onChange` | `(value: boolean) => void` | `value` 变化时的回调 |

### Data Attributes

| 属性 | 说明 |
| --- | --- |
| `[data-active]` | Present when active or pressed |
| `[data-focus]` | Present when focused |
| `[data-focus-visible]` | Present when focused with keyboard |
| `[data-readonly]` | Present when read-only |
| `[data-hover]` | Present when hovered |
| `[data-disabled]` | Present when disabled |
| `[data-checked]` | |

## FAQ

### 关于命名

> 作为输入器的一种，利用 **前缀命名法** 能够天然集合在一起。

有以下几种命名：

* `Switch` 多数选择
* `Toggle` 也有不少
* `InputSwitch` 只有 [primereact](https://primereact.org/inputswitch) 和 `kcuf-ui`
