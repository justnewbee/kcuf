# React + TypeScript + SWC

开发 React 组件的模版：

* 使用 [storybook](https://storybook.js.org) 本地开发调试
* 使用 [swc](https://swc.rs) 构建

## 命令

* `start` 启用本地开发（storybook）
* `prepublishOnly` 构建，在发布 npm 时执行
* `build:clean` 删除构建目录
* `build:esm` 构建 ESM
* `build:cjs` 构建 CJS
* `build:types` 构建类型
* `build:sb` 构建 storybook（暂时用不到）
